# STANDARD RESPONSE CONTRACT

Create:

types/api.ts

Implement:

```ts
export interface ApiError {
  code: string;
  message: string;
  details?: unknown;
}

export interface ApiResponse<T> {
  success: boolean;
  data?: T;
  error?: ApiError;
  requestId: string;
  timestamp: string;
}
```

Every endpoint MUST return this envelope.

No endpoint may return inconsistent JSON.
